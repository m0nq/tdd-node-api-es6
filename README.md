# tdd-node-api-es6

#### the practice of test driving node.js
---

## Getting Started##

#### Change the node version, clone the repo, install the modules, run lint and run the tests:####

````
nvm use v6.8.0
git clone git@github.com:wpannell/tdd-node-api-es6.git
cd tdd-node-api-es6
npm install
npm run lint
npm run spec
npm test
````
